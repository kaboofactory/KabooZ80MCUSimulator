# ⚡ KabooZ80 MCU Simulator

**究極の Web ベース Z80 開発環境 & バーチャルトレーナーキット**

(編注:このREADME.mdはAIに仰々しく書くように言いました。MITライセンスなのは本当です。)

## 🌟 概要

レトロコンピューティング教育の未来へようこそ！ **KabooZ80 MCU Simulator** は単なるエミュレーターではありません。これはブラウザ上で動作する、**超リアルで没入感のある、完全統合型の開発エコシステム**です。

熱狂的なファン、学生、そしてレトロウィザードたちのために設計されたこのプロジェクトは、伝説のマイクロプロセッサ「Z80」を、洗練されたローカライズ済み IDE とパーフェクトな仮想ハードウェアボード で現代に蘇らせます。現代の利便性を享受しながら、低レベルプログラミングのスリルを体験してください！

## 🚀 主な機能

### 💻 プロフェッショナルグレードのアセンブリ IDE
最先端の Web エディタで、プロのようにコードを書きましょう。
*   **インテリジェントなシンタックスハイライト**: ニーモニックをかつてないほど鮮明に読み取れます。
*   **インスタント・アセンブラ**: ASM コードをミリ秒単位でコンパイル。外部ツールは一切不要です。
*   **サンプルライブラリ**: 組み込みのサンプル（Hello World、電卓、ブロック崩しゲーム！）で、開発の旅をすぐに始められます。

### 🧠 深層デバッギング・ダッシュボード
マシンの内部動作を完全に可視化します。
*   **リアルタイム・レジスタ可視化**: コードの実行に合わせて踊るように変化する `AF`, `BC`, `DE`, `HL`, `IX`, `IY` を目撃してください。
*   **ライブ・メモリダンプ**: RAM のすべてのバイトをリアルタイムで検査可能。
*   **ステップ実行トレース**: サブルーチンに飛んでも迷子になることはもうありません。
*   **ブレークポイント制御**: 行番号をクリックするだけで、必要な場所で正確に実行を停止できます。

### 🎛️ 超リアル・バーチャルトレーナーキット
思わず触りたくなるほどリアルな、シミュレートされた物理ボードと対話してください。特に注目の **「Atmosphere Parts（雰囲気パーツ）」** セクションには、金属的な配線や、本物そっくりの抵抗器カラーコード（1kΩ や 680Ω の違いまで完全再現！）を備えたリアルな電子部品が搭載されています。

**搭載周辺機器:**
*   **🔴 8ビット LED アレイ**: バイナリカウントの視覚化に最適。
*   **🔢 8桁 7セグメントディスプレイ**: 数値データのマルチプレクス出力に対応。
*   **📟 16x2 LCD パネル**: フルアルファニュメリック出力をサポート。
*   **💡 8x8 ドットマトリックス**: 美しいアニメーションやグラフィックスを作成可能。
*   **🎹 4x4 メカニカルキーパッド**: 本格的な入力処理を実現。
*   **🎚️ DIP スイッチバンク**: ハードウェアレベルの設定シミュレーション。(編注:チャタリング非対応だから流石に嘘だろこれ)
*   **🔊 圧電ブザー**: チップチューンスタイルのオーディオ出力を生成。
